# Changelog

## [5.0.0] - 2025-08-23

### 🚀 Complete Rewrite - Radical Simplification

This is a complete rewrite of the Exocortex plugin, focusing on simplicity and core functionality.

#### What Changed
- **Removed 95% of features** to focus on essentials
- **Bundle size reduced from 362KB to 4.1KB** (99% reduction)
- **Only 3 source files** instead of 200+
- **Zero external dependencies** beyond Obsidian API

#### Features Kept
- ✅ Dynamic layout display of related assets
- ✅ Simple modal to create new assets

#### Features Removed
- ❌ All semantic web/RDF functionality
- ❌ All query engines and complex querying
- ❌ All graph visualization
- ❌ All testing infrastructure
- ❌ All mobile-specific features
- ❌ All API servers
- ❌ All dependency injection
- ❌ All agent systems
- ❌ All templates and configurations
- ❌ All button systems
- ❌ All property editing
- ❌ All task management
- ❌ All visual query features
- ❌ All autocomplete features
- ❌ All export/import features

#### For Users
If you need the advanced features, please use version 4.x. This version is for users who want a simple, fast plugin for basic asset management.
