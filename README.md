# Exocortex Plugin for Obsidian

A simplified Obsidian plugin that provides dynamic layouts and asset management.

## Features

- **Dynamic Layouts**: Display related assets grouped by relationship type
- **Asset Creation**: Simple modal to create new assets with metadata

## Usage

### Creating Assets

Use the command palette and search for "Create new asset" to open the asset creation modal.

### Using Layouts

Add a code block with `layout` to any note:

```markdown
```layout
```
```

This will display all related assets grouped by their relationship type.

## Installation

1. Download the latest release
2. Extract to your `.obsidian/plugins/` folder
3. Enable the plugin in Obsidian settings
